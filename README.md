# 📊 Data Analytics Project: Global Causes of Death Analysis

---

## 🔍 Overview

This project focuses on analyzing global causes of death from 1990 to 2017 using data analytics and visualization techniques.
The objective is to transform raw data into meaningful insights through an interactive dashboard, detailed report, and presentation.

---

## 📁 Dataset

The dataset includes:

* Year-wise data on various causes of death
* Categories such as communicable, non-communicable, lifestyle, and environmental factors

---

## 🛠 Tools & Technologies

* **Power BI** – Dashboard development
* **DAX** – Calculated measures and KPIs
* **Power Query** – Data transformation
* **Gamma AI** – Presentation (PPT) creation

---

## ⚙️ Project Steps

1. Data cleaning and transformation using Power Query
2. Data modeling and creation of calculated measures (DAX)
3. Building interactive dashboards in Power BI
4. Extracting insights and summarizing findings
5. Creating a report and presentation using Gamma AI

---

## 📊 Dashboard

The Power BI dashboard includes:

* Key performance indicators (KPIs)
* Trend analysis over time
* Top causes of death
* Category-based comparison (Communicable vs Non-Communicable)
* Lifestyle vs Environmental analysis

---

## 📈 Results & Insights

* Non-communicable diseases account for the majority of deaths
* Cardiovascular diseases are the leading cause globally
* Lifestyle-related factors contribute significantly more than environmental causes
* Death trends show a consistent increase over time

---

## ▶️ How to Run

1. Download the "causes of death.pbix" file from this repository
2. Open it using Power BI Desktop
3. Interact with filters and visuals to explore insights
